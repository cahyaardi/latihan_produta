# 🌿 Supply Chain Agrinesia — Streamlit App

Prototipe sistem Supply Chain & Logistik PT Agrinesia Raya untuk Tugas Akhir.

## 🚀 Cara Menjalankan

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Jalankan aplikasi
```bash
streamlit run app.py
```

### 3. Buka browser
Aplikasi akan terbuka di `http://localhost:8501`

---

## 📋 Fitur

| Modul | Deskripsi |
|---|---|
| 🏠 Home | Dashboard KPI & grafik ringkasan |
| 📦 Peramalan Permintaan | Forecasting dengan confidence interval |
| 🌡️ Prediksi Suhu | Model steamflow → suhu steam tunnel |
| 🏭 Pengendalian Mutu | Control chart & DMAIC visual |
| 🚚 Optimasi Distribusi | Simulasi rute & biaya pengiriman |

## 🛠️ Teknologi
- **Python 3.9+**
- **Streamlit** — UI framework
- **Plotly** — visualisasi interaktif
- **Pandas / NumPy** — data processing
